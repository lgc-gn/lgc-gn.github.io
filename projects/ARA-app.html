Active Reading Assistant (ARA)

GitHub Repo

    The ARA Project was designed during my Software Methodologies class. Alongside 4 other group mates, we designed and created a stand-alone application that supports notetaking using the SQ3R reading style. In charge of backend, I handled server-side storage using MongoDB to ensure notes could be saved across devices, provide 3 default users to save/load notes from, and custom user creation.
    Tools Used:
  1. Python
  2. MongoDB
