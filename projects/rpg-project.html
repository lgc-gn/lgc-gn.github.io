Untitled RPG Game

GitHub Repo additionally, play it here


The Untitled RPG Game was a game project I lead during my junior year for a game development course. Inspired by games such as Final Fantasy Tactics: War of the Lions and Baldur's Gate, my group and I decided to make a turn based combat game with chess-like movement, and custom classes inspired by the Dungeons & Dragons universe.

In the span of 10 weeks we developed:
    A tile-based movement/combat system
    Simple enemy AI for the player to defeat
    Custom user interface to allow player control and provide information
    Animation blending and event triggered effects
    Class skills that match their themes
We used Trello to manage and delegate tasks to group members, which can be foundhere
    Tools used:
  1. Unity
  2. C#
  3. Git
